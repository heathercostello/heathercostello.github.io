JavaScript Objects

Object Creation Functions, Inheritance, Properties, Methods, Instantiation

Objects are everything in JavaScript. If you can understand objects then you will be able to understand JavaScript.

In JavaScript, almost everything is an object. All JavaScript values, except primitives, are objects.

  • Booleans can be objects (if defined with the new keyword)
  • Numbers can be objects (if defined with the new keyword)
  • Strings can be objects (if defined with the new keyword)
  • Dates are always objects
  • Maths are always objects
  • Regular expressions are always objects
  • Arrays are always objects
  • Functions are always objects
  • Objects are always objects

Creation

Object Initializers -used to create objects. There are three different ways. Spaces do not matter.

var car = {
carMake: "Ford",
carModel: "F150",
carYear: "2013"
};

var car = { carMake: "Ford", carModel: "F150", carYear: "2013"};

var car = new Object();
car.carMake = "Ford";
car.carModel = "F150";
car.carYear = "2013";

Properites

A collection of properties are basically whatmakes up an object. A property is made up of a key (name) and a value. A dictionary is a great example of this. The dictionary represents an object, the term represents the key, and the definition represents the value. Properties are used to define the characteristic of the object.

To access the property, we use this syntax
objectName.propertyName

It is easy to add and delete properties.

Example of creating an object, adding properties, and then deleting a property.

*creating a new object

var arizonaStaples = new Object ();
*adding three new properties
arizonaStaples.food = "Bean Burritos";
arizonaStaples.weather = "Sunny";
arizonaStaples.landscape = "Cactus";
*deleting one property
delete arizonaStaples.weather;

Inheritance

All JavaScript objects inherit properties and methods from a prototype. The Object.prototype is on top of the prototype inheritance chain. Once the prototype has been created, newinstances of the object can be made. The object can then be changed by adding new properties or methods and then ahve new objects created from it. It is cool because it allows code to be reused without it having to be written again.
