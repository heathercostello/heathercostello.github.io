JSON

JSON Parse, Stringify

JSON means JavaScript Object Notation
JSON is a syntax for storing and exchanging data.
JSON is text, written with JavaScript object notation

JSON is awesome because it allows data to be sent to a server or be recieved from a server. If the data is stored in a JavaScript object then it will neet to be converted into JSON. If you recieve data in JSON format, you can convert it into a JavaScript object.

JSON.parse()

JSON.parse() creates data that comes in from a web server as a string into a JavaScript object
