JavaScript Loops

Loops are used to execute a block of code over and over while a certain condition applies.

  • For Loops -Used when you know in advance how many times the script should run
  • For/In Loops -Loops through the properties of an object
  • While Loops -Used when you want to loop to execute and continue executing while the specified condition is true
  • While Do Loops -Will execute the code block once, before checking if the condition is ture and then it will repeat the loop as long as the condition is true.

Conditional Statements

Very often when you write code, you want to perform different actions for different decisions.
You can use conditional statements in your code to do this.

  • Use if to specify a block of code to be executed, if a specified condition is true
  • Use else to specify a block of code to be executed, if the same condition is false
  • Use else if to specify a new condition to test, if the first condition is false
  • Use switch to specify many alternative blocks of code to be executed

Example of a conditional statement

We use conditional statements in our lives every day, for example, when one turns 16 they can get their license IF they have had their permit for six months and are at the age of 16. Lets put this in a function.

Can you get your license?
How old are you?
How many months have you had your permit for?

Functions, Variables, and Parameters

A Function is a block of code designed to perform a particular task. It can only be executed when something calls it. There are a variety of different types of functions.

Variables are containers for storing data values

var day1 = 5;
var day2 = 6;
var total = day1 + day2;

Parameters are containers for storing data values

Function parameters are the names listed in the function definition.

Parameter Rules

  • JavaScript function definitions do not specify data types for parameters.
  • JavaScript functions do not perform type checking on the passed arguments.
  • JavaScript functions do not check the number of arguments received.

Associative Arrays

Associative Arrays are dynamic objects that the user redefines as needed. When you assign values to keys in a variable of type Array, the array is transformed into an object, and it loses the attributes and methods of Array. The length attribute has no effect because the variable is not longer of Array type.
